Financial Ratios Go

Financial Ratios Go is a financial ratio calculator.

It can fetch financial statements of 2000+ major companies by using API from https://financialmodelingprep.com/

It fetches company icons from https://clearbit.com/docs#autocomplete-api

It can also store user-defined financial statements and add fiscal period.

The current available financial ratios are:

  1. Liquidity Measurement Ratios : Current Ratio, Cash Ratio
  2. Debt Ratios : Debt Ratio, Interest Coverage, Cash Flow to Debt Ratio,
  3. Profitability Indicator Ratios : Operating Profit Margin, Net Profit Margin, Return on Equity, Return on Debt, Return on Invested Captial
  4. Cash Flow Indicator Ratios : Cash Flow Return on Investment (CFROI), Divident Payout Ratio

We will add more financial statements/ratios in the future.
